> [!Abstract]
> This page provides a general overview of the custom software being developed for Coliving Liguria. Our goal is to create a bespoke, integrated suite to manage the unique operational needs of our co-living community, from resident registration to financial forecasting.

### Software Projects

This repository contains two primary software projects: the **Resident & Property Register** and the **Financial Management Tool**.

#### 1. Resident & Property Register

A Python-based application for managing all logistical data related to our properties and residents.
* **Status:** In Development (Core functionality implemented)
* **Description:** This tool is designed to be the central database for all non-financial information, managing resident registration, property details, guest types, and basic payment information.
* **Future Work:** The roadmap includes migrating to a robust database (like PostgreSQL), developing a user-friendly graphical interface (UI), and refactoring the code for better maintainability.

#### 2. Financial Management & Forecasting Tool

A more ambitious project designed to handle all financial aspects of the co-living community, from tracking resident payments to providing predictive insights.
* **Status:** Planned (Conceptual Phase)
* **Description:** This software will serve as the financial brain of the operation, integrating seamlessly with the Register. It will manage all financial data, generate reports, and include a model for tracking revenue and making future predictions.

### Get Involved

The project's code is hosted on GitHub: [https://github.com/Toxytod/ColivingLiguria](https://github.com/Toxytod/ColivingLiguria)

The repository is currently private, but anyone willing to contribute to the software development is welcome. Just get in touch ([[Contact]]) to be added to the team!

> [!Abstract]
> Questa pagina fornisce una panoramica generale del software personalizzato in fase di sviluppo per ColivingLiguria. Il nostro obiettivo è creare una suite integrata e su misura per gestire le esigenze operative uniche della nostra comunità di co-living, dalla registrazione dei residenti alla previsione finanziaria.

### Progetti Software

Questa repository contiene due progetti software principali: il **Registro dei Residenti e delle Proprietà** e lo **Strumento di Gestione Finanziaria**.

#### 1. Registro dei Residenti e delle Proprietà

Un'applicazione basata su Python per la gestione di tutti i dati logistici relativi alle nostre proprietà e ai residenti.
* **Stato:** In Sviluppo (Funzionalità principali implementate)
* **Descrizione:** Questo strumento è progettato per essere il database centrale per tutte le informazioni non finanziarie, gestendo la registrazione dei residenti, i dettagli delle proprietà, le tipologie di ospiti e le informazioni di base sui pagamenti.
* **Sviluppi Futuri:** La roadmap include la migrazione a un database robusto (come PostgreSQL), lo sviluppo di un'interfaccia grafica (UI) intuitiva e il refactoring del codice per una migliore manutenibilità.

#### 2. Strumento di Gestione Finanziaria e Previsionale

Un progetto più ambizioso, pensato per gestire tutti gli aspetti finanziari della comunità, dal tracciamento dei pagamenti dei residenti alla fornitura di analisi predittive.
* **Stato:** Pianificato (Fase concettuale)
* **Descrizione:** Questo software fungerà da "cervello" finanziario dell'operazione, integrandosi perfettamente con il Registro. Gestirà tutti i dati finanziari, genererà report e includerà un modello per monitorare i ricavi e fare previsioni future.
